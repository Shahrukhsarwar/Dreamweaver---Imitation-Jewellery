# eProject: Pascal Jewellery (Single Page Website)

## 📌 Project Overview
This repository contains the complete source code, compiled production assets, and full technical documentation for the **Pascal Jewellery** Single Page Website. This project has been developed as part of the Semester-End eProject evaluation.

**Pascal Jewellery** is a premium, luxury imitation jewellery brand designed to showcase artificial, semi-precious, and luxury jewellery items crafted with elegant finishes for clients across Pakistan.

* **Project Start Date:** 16-Apr-2026  
* **Project End Date:** 16-May-2026
